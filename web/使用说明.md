# 命运回响 · 网页离线版

这个版本面向没有计算机基础的用户。

## 怎么运行

双击打开 `index.html`。

不需要安装 Python，不需要命令行，不需要虚拟环境，也不需要联网。

## 怎么发给别人

把整个 `web` 文件夹发给对方即可。至少包含：

- `index.html`
- `style.css`
- `app.js`
- `story-data.js`

## 怎么变成真正的网址

把 `web` 文件夹里的四个文件上传到任意静态网站托管服务即可，例如：

- GitHub Pages
- Netlify
- Vercel
- 腾讯云 COS 静态网站
- 阿里云 OSS 静态网站

上传后，对方只需要打开链接。

## 和微信小程序相比

网页离线版更适合快速分发，因为它不需要微信开发者工具、AppID、审核流程或小程序发布权限。

如果后续确定要做微信小程序，可以复用 `story-data.js` 里的剧情数据，再单独开发小程序页面。

## AI 功能怎么用

网页右上角点击 `AI设置`，再点击 `填入 DeepSeek 预设`。

然后把你自己的 DeepSeek API Key 粘贴进去并保存。Key 只会保存在当前浏览器的 localStorage 中，不会写入项目文件。

DeepSeek 官方 Chat Completions 文档地址：<https://api-docs.deepseek.com/api/create-chat-completion>

AI 功能可用于：

- 开局时优化人设
- 在人生节点中生成一条额外选择

注意：纯网页直接调用大模型接口可能遇到浏览器跨域限制。如果出现请求失败，说明需要增加后端代理，或者继续使用本地随机功能。

不要把写死 API Key 的网页公开发布。公开网页应由后端保存 Key。
