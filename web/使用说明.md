# 命运回响 · 网页离线版

这个版本面向没有计算机基础的用户。

## 怎么运行

双击打开 `index.html`。

不需要安装 Python，不需要命令行，不需要虚拟环境，也不需要联网。

## 怎么发给别人

把整个 `web` 文件夹发给对方即可。至少包含：

- `index.html`
- `style.css`
- `app.js`
- `story-data.js`

## 怎么变成真正的网址

把 `web` 文件夹里的四个文件上传到任意静态网站托管服务即可，例如：

- GitHub Pages
- Netlify
- Vercel
- 腾讯云 COS 静态网站
- 阿里云 OSS 静态网站

上传后，对方只需要打开链接。

## 和微信小程序相比

网页离线版更适合快速分发，因为它不需要微信开发者工具、AppID、审核流程或小程序发布权限。

如果后续确定要做微信小程序，可以复用 `story-data.js` 里的剧情数据，再单独开发小程序页面。
