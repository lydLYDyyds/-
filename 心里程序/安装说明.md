# 安装说明

## 最简单用法

双击 `一键启动.bat`。

脚本会自动完成：

- 创建本地 `.venv` 运行环境
- 安装依赖
- 打开浏览器
- 启动 Streamlit 应用

## 如果电脑没有 Python

1. 打开 https://www.python.org/downloads/
2. 下载 Python 3.10 或更高版本。
3. 安装第一页勾选 `Add python.exe to PATH`。
4. 安装完成后，重新双击 `一键启动.bat`。

## 如果想打包成 exe

双击 `一键打包EXE.bat`。

打包结果会在 `dist\命运回响` 文件夹中。把这个文件夹整体发给别人，对方不需要安装 Python，也不需要手动安装依赖。

注意：由于 Streamlit 本身是 Web 应用，exe 仍会在本机启动一个浏览器页面。
