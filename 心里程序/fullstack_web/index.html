风格摘要

    AI 设置

    API Key 保存在本机后端的 `life_sim_data/ai_config.json`，不会暴露在前端 JS 中。公开发布时仍建议做更严格的后端权限保护。

    使用说明

    这是面向本地运行的完整前后端版本：Python FastAPI 后端负责剧情、随机事件和大模型调用；前端只负责展示和交互。

    • 打开首页后先选择人设，再进入剧情。
    • 每章可能出现随机状态事件，选项也可能出现本地动态选项。
    • 配置 AI 后，可以优化人设，也可以在每章生成一条 AI 额外选择。
    • 如果 AI 返回 402，通常是 API 余额不足。
